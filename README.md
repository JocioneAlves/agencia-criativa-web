# Agência Criativa Web

Projeto desenvolvido como parte do curso da EBAC.

## 🚀 Sobre o projeto

Este é um site institucional de uma agência criativa, contendo:

- Home
- Sobre
- Serviços
- Depoimentos
- Contato

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3

## 🎯 Refatoração

O projeto foi refatorado utilizando a metodologia BEM (Block, Element, Modifier), com foco em:

- Melhor organização do CSS
- Redução de especificidade
- Código mais limpo e reutilizável
- Facilidade de manutenção

## 📱 Responsividade

O layout é totalmente responsivo, adaptando-se a diferentes tamanhos de tela.
